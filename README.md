* Simple, clean UI

---

## Purpose

This project was built to:

* Learn **YouTube video integration** in Flutter
* Practice **package usage and state management**
* Create a functional, real-world app in minimal code

---

## Tech Stack

* Flutter
* Dart
* `youtube_player_flutter` package

```bash
git clone <repo-url>
```
